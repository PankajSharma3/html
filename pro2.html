Motherboard
The Motherboard is a Printed Circuit Board that every other computer hardware component will be attached to in one way or another. It functions like a central hub that manages and connects all of your other parts.
CPU
CPU means Central Processing Unit and it processes any task that you might throw at it. Without it, the computer won,t work.
Almost everything you do on a computer will have to be calculated by the CPU in some way, so having a fast CPU (high frequency and high core count) will make your PC faster.
RAM
Random Access Memory (RAM) is the “short-term” memory part of the computer if you would compare it to a brain.
he RAM stores data that is actively being worked on by the CPU. RAM can read and write data extremely fast (much faster than any permanent storage solutions) but can't retain information once the power is turned off.
STORAGE DEVICE
Because RAM can't store anything without power and we do want to be able to turn off our computer from time to time, we need a storage medium that retains its stored data, even without electricity.
PSU
We went through many of the most important PC Components already, but your PC won’t work without a power supply to supply the PC with power.
GPU
Its purpose is the processing of anything that has to do with visuals and outputs these visuals (Images, User Interface, GUI, Video) onto the Monitor(s).
CASE
The Computer case is nothing more than a fancy-looking box that holds all of the PC’s components. It can be opened and closed and has pre-defined areas with screws and holes where all of the components are supposed to be placed and attached to.
COOLING

MONITOR
Any PC that you want to interact with needs a display device such as a monitor, to be able to see what’s going on. Monitors come in all kinds of sizes, colors, resolutions, aspect ratios, and so on.
INPUT DEVICE
The devices which we plug into our sysytem , such as keyboard,mouse etc..
