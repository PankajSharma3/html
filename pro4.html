Drivers

Every Physical components that we have used to build the computer,require latest drivers to run efficiently.so we are gonna install the required drivers.


Components require drivers
  1. CPU
  2. GPU
  3. Others
CPU
CPU aka Central Processing Unit.
We can get the driver required for CPU from the CPU vendor itself.
By going to the driver section of your CPU which u have used.
Download the required softwares(recommened to download only stable version).
GPU u have one
GPU aka Graphics Processing unit
Same like CPU, we have to get the drivers from vendor itself.
we can get the drivers from the vendor's website
Others
Others include the components present in your Motherboard i.e wifi adapter,bluetooth,usb plugins etc..
Same like above we will be able to get all the latest required drivers from the your Motherboard vendor website.
